# CEMENTED — Website

Premium grooming subscription brand. Built with HTML, CSS, and vanilla JS.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main HTML structure |
| `styles.css` | All styles, animations, responsive |
| `main.js` | Quiz overlay, cursor, scroll animations, interactions |

## Launch on GitHub Pages

1. Create a new repo on GitHub (e.g. `cemented-website`)
2. Upload all 3 files to the root of the repo
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Select branch: `main`, folder: `/ (root)`
6. Click **Save**
7. Your site will be live at: `https://yourusername.github.io/cemented-website`

## Custom Domain (optional)

To use `www.getcemented.com` or similar:
1. Add a `CNAME` file to the repo containing just your domain: `www.getcemented.com`
2. Go to your domain registrar and add a CNAME DNS record:
   - Name: `www`
   - Value: `yourusername.github.io`
3. Back in GitHub Pages settings, enter your custom domain and check **Enforce HTTPS**
